* Hier ga ik mijn eerste Angular app ontwikkelen aan de hand van de [Angular-tutorial](https://angular.dev/tutorials/first-app).

* Install Angular 

`npm install -g @angular/cli`

* Make a folder

`cd my-app`

* Install the depencies

`npm install`

* Run the application

`ng serve`
